# 위드미 (WithMe)

위드미는 인공지능 기반 시각 장애 보조 어플리케이션 서비스입니다.

위드미는 [위드미 레거시](https://github.com/DY-MAKE/WithMe-legacy) 프로젝트를 기반으로 하는 새 위드미 프로젝트입니다.
